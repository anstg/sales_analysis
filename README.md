# sales_analysis
Проект на Python для генерации и анализа продаж

## Функционал
- Генерация продаж за 12 месяцев для нескольких клиентов и продуктов
- Расчет средних трат по возрастным группам
- Определение топ-5 продуктов по доходу и их доля
- Визуализация



## Использование

1. Генерация данных:

```bash
python generate_data.py

 2. Аналитика и визуализация:

python analyze_sales.py

 3. Тестирование:

python tests.py
